# Senior Living Operations Intelligence
A portfolio project demonstrating unified analytics across fragmented public data sources in the senior living and skilled nursing space.
## What this is
Senior living and skilled nursing operators sit on data scattered across 8–12 systems — EHR, billing, scheduling, compliance, surveys, and more. Cross-system questions take weeks because no one owns the seams.
This project unifies the public data landscape (CMS, BLS, HRSA, Census, and state licensing data) to demonstrate the same identity-resolution and unified-modeling pattern that works on operators' internal stacks.

🚧 Work in progress. Currently loading and exploring CMS Provider Info and Health Deficiencies data. Peer-cohort staffing dashboard coming next.
